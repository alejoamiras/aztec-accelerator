How it works

Install once. Every Aztec dApp on your machine gets native-speed proving automatically.

1

Install

Download the desktop app. It sits in your system tray and manages bb proving binaries automatically.

2

Run silently

The accelerator runs in the background. No windows, no interruptions — just a tray icon showing status.

3

Apps get faster

Any dApp using the SDK auto-detects the accelerator and proves natively instead of through browser WASM.

Safari users

Safari blocks mixed-content requests to localhost. The accelerator includes an optional local HTTPS certificate so it works seamlessly in Safari too. You'll be prompted to authorize it during setup.
