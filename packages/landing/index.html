How it works

Install once. Every Aztec app on your machine gets native-speed proving automatically.

1

Install

Download the desktop app. It lives in your system tray and manages proving binaries automatically.

2

Stays in your tray

No windows, no interruptions. The accelerator stays in your menu bar with a tray icon showing real-time status.

3

Apps get faster

Any app using the SDK detects the accelerator and proves natively instead of through browser WASM.

Safari users

Safari requires HTTPS for localhost connections. On first launch, the accelerator generates a local-only certificate and asks macOS to trust it once. After that, Safari works automatically. Your keys never leave your machine.
